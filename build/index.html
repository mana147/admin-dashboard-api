API Upload file :

Upload file lên hệ thống MPS
- Nếu upload từ server dự án, có thể upload sử dụng secret key.
- Nếu upload từ client, có thể sử dụng policy signature.
- Hiện hệ thống giới hạn dung lượng file tối đa là 8GB.

Endpoint : GET < API_ENDPOINT>/_/ls

Tham_số Bắt_buộc Loại Nguồn Mô_tả
secret_key Có String Query string Secret key của phân vùng lưu trữ
dirpath Có String Query string Thư mục của namespace muốn liệt kê file
page Không Int Query string Kết quả trả về được phân trang

Kết quả trả về 

                            
{
    'directories': mảng danh sách thư mục con cấp 1 của thư mục muốn liệt kê,
    'files': mảng danh sách file nằm trong thư mục muốn liệt kê,
    'page': 'trang/tổng số trang kết quả
}
Mã lỗi Mô tả
secret_key Có
dirpath Có
page Không
